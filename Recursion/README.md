# Recursion

### In 4 Steps:

1. Identify base case(s)
2. Identify recursive case(s) _i.e.work that you want to do_
3. Return where appropriate
4. Write procedures for each case that bring you closer to the base case(s)

```js
const cat = () => {
  if () {
    // base case
    return;
  } else {
    // recursive case
    cat();
  }
  return;
};
```
